# AI Web Scout — прототип интерфейса (Clay-стиль)

Интерактивный прототип нового UI сервиса в табличном формате, как в Clay:
**домены — в левом столбце, AI-промпт-столбцы — справа**, результат по каждому
домену в ячейке. Это **только дизайн интерфейса** (фронтенд-макет с моковыми
данными), без подключения к реальному бэкенду.

## Как посмотреть

Открыть файл в браузере:

```
prototype/scout-clay-ui.html
```

Просто двойной клик по файлу или:

```bash
# из корня репозитория
python3 -m http.server 8000
# затем открыть http://localhost:8000/prototype/scout-clay-ui.html
```

Один self-contained HTML-файл, без сборки и зависимостей.

## Что можно потрогать

- **Запустить столбец** — кнопка «Запустить» в шапке столбца. Показывает все
  3 стадии текущего пайплайна: `Ключевые слова → Краулинг → LLM-извлечение`,
  ячейки заполняются по очереди (found / ❌ не найдено + source_url).
- **Запустить всё** — прогоняет все столбцы сразу.
- **Добавить столбец** (`+ Столбец`) — выбор типа (AI-промпт / скрейпинг /
  обогащение / формула), промпт на весь столбец с переменными
  `{{Company Name}}` и `{{Website}}`, выбор модели, готовые шаблоны.
- **Редактировать промпт** — клик по строке промпта в шапке столбца.
- **Клик по ячейке** — боковая панель с результатом, source_url, промптом,
  моделью и «следом краулинга» (какие страницы обошёл робот).
- **Импорт доменов** — вставка списка (по одному на строку), как сейчас.
- **Экспорт** — CSV (реально скачивается) и макет выгрузки в Google Sheets.
- **Меню столбца** (⋯) — запуск / редактирование / дублирование / удаление.
- **Поиск** по доменам, тёмная тема (иконка солнца).

## Что отражено из текущего сервиса

| Текущая возможность | Где в прототипе |
|---|---|
| Загрузка списка доменов | Левый столбец + «Импорт доменов» |
| Запрос «что ищем» (free-form) | Промпт на весь столбец |
| Шаблоны `{{Company Name}}`, `{{Website}}` | Чипы-переменные в редакторе промпта |
| Stage 1 — генерация ключевых слов | Полоса прогресса пайплайна + чипы ключевых слов |
| Stage 2 — краулинг (5 стр./домен) | Стадия 2 + «след краулинга» в панели ячейки |
| Stage 3 — LLM-извлечение (GPT-4o-mini) | Стадия 3, бейдж модели на столбце |
| Результат: found / ❌ Не найдено + source_url | Содержимое ячеек и боковая панель |
| OpenAI API Key, воркеры, лимиты страниц | Модалка «Настройки» |
| Экспорт CSV / Google Sheets | Модалка «Экспорт» |

## Дизайн-система (из скилла ui-ux-pro-max)

- Палитра: teal `#0D9488` (основной) + orange `#EA580C` (акцент/CTA)
- Стиль: micro-interactions, плотная data-table вёрстка
- Светлая и тёмная темы, доступные контрасты, hover/focus-состояния
