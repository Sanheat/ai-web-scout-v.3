# AI Web Scout — Clay-style UI prototype

Интерактивный **макет** нового интерфейса в эстетике Clay. Это автономный
HTML-файл с моковыми данными (без бэкенда и без обращения к OpenAI) — нужен,
чтобы «потыкать» концепцию до переноса реального сервиса.

## Как открыть
Просто откройте `index.html` в браузере (двойной клик). Ничего ставить не нужно.

## Идея
- **Строки** — домены (как в текущем `app.py`, но теперь это «сущности»).
- **Столбцы** — независимые промпты/ТЗ для нейросети. Каждый столбец
  обогащает все домены своим контекстом. Столбцов можно добавлять сколько угодно.

## Что можно потыкать
- **＋ Столбец** — добавить промпт-столбец из шаблона или с нуля (модель, иконка,
  промпт с переменными `{{Website}}`, `{{Company Name}}`).
- **Клик по заголовку столбца** — настроить/переписать промпт.
- **⋯ на заголовке** — запустить / дублировать / очистить / удалить столбец.
- **▶ Запустить все** — имитация обогащения: ячейки уходят в «анализ…», затем
  заполняются значением + ссылкой на источник (или «не найдено»).
- **Клик по ячейке** — боковая панель с результатом, источником и промптом.
- **＋ Добавить домен** — новая строка.

## Соответствие текущему коду
| Сейчас (`app.py`)            | В прототипе                          |
|------------------------------|--------------------------------------|
| textarea со списком доменов  | строки таблицы (левый столбец)       |
| один общий «Что ищем?»        | множество промпт-столбцов            |
| `llm_validator.validate_with_llm` | логика одного столбца           |
| плоская `results.csv`         | сетка домен × промпт                 |

> Это только фронтенд-макет. Данные сгенерированы на клиенте для демонстрации.
